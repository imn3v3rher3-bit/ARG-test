How should you protect yourself?
  1. Verify the source
    2. Ask yourself; Is the website legitimate?

    You can use a website legitimacy checker in any browser. In Safari or Edge, lookout for a padlock next to the URL to ensure you have a secure connection to the website. In Chrome, it alerts you when it detects a site that may be unsafe to visit.

  2. Avoid sharing personal information, such as:
    • Password
    • Full name
    • Address
    • Phone number
    • Personal details

  3. Avoid clicking on links or downloading applications

    1. Use a Link Checker Tool
    2. Make sure the site uses HTTPS, the data sent between your browser and the website is encrypted
    3. Check the Domain Age and Ownership
    4. Have good antivirus software installed, such as Norton 360, McAfree, etc.

  5. Have a strong password

    6. A strong password comprises of:

    • Length: At least 12 to 16 characters long
    • Complexity: Combine upper and lower-case letters, numbers, and special characters
    • Uniqueness: Do not use personal information, easy words or predictable sequences
    • Memorability: Easy to remember but difficult for others to guess
